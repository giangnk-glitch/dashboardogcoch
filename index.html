Hướng dẫn kết nối Google Sheets

Dashboard này đọc dữ liệu qua Google Sheets Published CSV — miễn phí, không cần backend, không cần API key.
Bước 1 — Publish Sheets của anh
  1. Mở file Google Sheets → File → Share → Publish to web
  2. Chọn từng sheet cần dùng → chọn định dạng Comma-separated values (.csv)
  3. Click Publish → copy link
  4. Link sẽ có dạng: https://docs.google.com/spreadsheets/d/SHEET_ID/pub?gid=GID&single=true&output=csv
Bước 2 — Điền Sheet ID và GID

Sheet ID của anh là: 1evs_g8zR1UfGcAzvDI4ZvLCQWPiMCw6IGrK55rmjxJc

Lấy từ URL: .../spreadsheets/d/ĐÂY/edit
Lấy từ URL khi đang ở tab đó: ...#gid=ĐÂY

Hướng dẫn — Publish Google Sheets

  1. Mở Google Sheets → File → Share → Publish to web
  2. Ở dropdown Entire document, chuyển sang chọn từng sheet cụ thể
  3. Định dạng: chọn Comma-separated values (.csv)
  4. Bấm Publish → xác nhận OK
  5. Link sẽ có dạng: …/pub?gid=GID&single=true&output=csv — copy phần số GID
  6. Làm lần lượt cho từng sheet cần dùng, dán GID vào các ô bên trên
